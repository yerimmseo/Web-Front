# CDN (Content Delivery Network)

  • 컨텐츠를 네트워크를 통해 제공하는 것
  • 컨텐츠 제공 회사가 제공하는 여러가지 정적 자원들 (글꼴, 아이콘, 이미지, JS, CSS 등)을 가져다 사용할 수 있다

# google icons (Material icon)

아이콘은 실제로는 글꼴이기 때문에 글자에 대한 스타일이 모두 적용된다.
※ 여러가지 타입의 구글 아이콘을 적용할 때는 link태그의 uri에 |를 이용한다

  • 좋아요favorite face
  • loyalty cruelty_free
  • sentiment_very_satisfied nights_stay icecream
